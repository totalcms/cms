# Total CMS 3.1 - Marketing Features

## Core Features

### No Database Required
Store all your content in simple JSON files—no MySQL, no server configuration, just files you can version control and deploy anywhere.

### 15+ Built-in Content Types
Blog posts, galleries, files, emails, and more—ready to use out of the box with zero configuration.

### Custom Collections
Create any content structure you need with 20+ field types, from simple text to complex repeatable sections.

### Visual Form Builder
Design content entry forms with a drag-and-drop interface—no coding required.
